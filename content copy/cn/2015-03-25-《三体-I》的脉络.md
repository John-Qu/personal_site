《三体》的脉络
2015.03.25
= 类别 =
    《三体》是刘慈欣作，属于科幻小说（SF：Scientific Fiction）。科幻小说类的特点，是有个基本的科学假设，在这个假设之下，通过故事的发展，探讨一些特殊的矛盾。
= 主题 =
    《三体》的基本假设是：如果外星文明的存在得到证实，他们是什么样子？我们会有哪些反应？以及地球与外星文明的接触将怎样发展？
= 线索 =
    《三体》的故事线索有三条。一个线索是现实的危机。基础科学的顶尖科学家相继自杀，尖端纳米材料的研究者受到神迹的恐吓，各国成立作战中心，放下隔阂，团结一致保卫地球文明。另一个线索是历史的运动。在彼此竞赛的环境下，人类用有限而渺茫的手段向外星文明喊话，红岸基地是代表之一。而世界的黑暗面与命运的残酷，造成了一些人对人类整体的失望或绝望。他们梦想外星文明也许是个解决的途径。第三个线索是三体的世界。三体是个经典的物理数学难题，而三体游戏所揭示的文明演化历程，也逐渐将这毁灭与重生的矛盾推向高潮。
= 高潮 =    
    全书的高潮是三体世界发给地球的信息解读。原来他们的文明有这样的能力，原来他们的文明对地球文明的看法与对策是这样子的。这解释了现实的危机产生的原因，打破了红岸为代表的同外星文明握手的梦幻。高潮之后，表现的是科学家的绝望和叶文洁的失望。
= 人物 =
    《三体》的主要人物与特征如下。
    汪淼——纳米材料科学家，全书贯穿情节的人物。人缘巨好，个性软弱平淡。
    叶文洁——天体物理学家，文化大革命中多次伤心，精神世界迷茫无助，不惜牺牲一切，机会面前幻想求助于外星文明。
    杨冬——叶文洁之女，纯洁的高能物理学家。被外星文明的颠覆策略摧毁的一朵美丽的花。
    丁仪——高能物理学家，发现宏原子，杨冬的男友。物理哲学精湛，汪淼的答疑解惑之师。
    申玉菲——“科学边界”的联络员，曾入降临派核心，后投身于拯救派。
    魏成——懒懒的数学天才，申玉菲的丈夫。百无聊赖中想到三体问题，爱情一般不可自拔。
    大史——老刑警，擅长看透人，乐观派，说“太邪乎必有鬼！”
    常思伟——中国“作战中心”的将军，大史的老上司，曾在对越自卫反击战中任侦查连（特种兵）连长。
    伊文斯——跨国石油公司继承人，物种共产主义践行者，对人类文明彻底失望，三体叛军降临派的领袖。
    潘伟——环保主义者，利用智子的知识给自己造名声。降临派力量在中国的代表。
    叶泰哲——叶文洁的父亲，老物理学家，文革中不肯低头，被小将当众殴打致死。
    邵琳——叶文洁的母亲，聪于世故的物理教师。
    杨卫宁——叶文洁父亲的学生，红岸基地的总工程师，叶文洁的丈夫。
    雷政富——红岸基地的政委，利用叶文洁的知识搏自己的名声。
= 文明问题 =
    == 谁的文明 ==
    以人为本的人类文明观，与更广大的地球文明观，与更广大的宇宙演化观，孰是孰非？ 雷军说：人欲即天理，这是更现实的价值观。
    有没有宇宙普适的善恶观？
    == 问题必然，救赎可能 ==
    文明发展演化中的负面问题，能否通过自我救赎来解决。（自己拉着头发拔离地球表面。）罗胖等乐观派的观点是，发展中的问题必能在发展中解决。
= 科学问题 =
    == 高维看低维 ==
    我们生活的世界貌似只到三维。三维的我们看二维的圆，可以不走平面，从空中进入。如果有更高的维度，他们看我们的三维，是否也可以不走三维，从空中进入？
    微观粒子的奥秘不断被揭示出来，他们貌似简单的外表之下，是不是包含着更高维度的复杂？一幅清明上河图包含的信息量，远远不如一片白云清空。简单的表象，隐藏着无尽的本质。
    文明水平的标志，是对维度（视角能力）的掌握。
    == 强者回话的放大效果 ==
    太阳气海有隔膜。向太阳发射电磁波，就像小声骂贵族。你虽然是屌丝，但骂的强度足够激怒贵族的话，他回骂的声音就是贵族的气派了。
    == 宇宙规律在时间和空间上均匀吗 ==
    虽然后来确认是三体世界发来的智子在搞乱，但是那些事件，能不能引起我们的反思。我们所追求的真理，数学能解释的范围，科学能探究的范围，是不是有边界？宇宙间是不是有更多的部分并不遵循我们所发现的规律？如果在小小的地球范围内，这些规律在不同时间不同地点的结果都不再一样，那么这些还能称为规律吗？科学还存在吗？如此说来，我们的很多努力也就没有价值了。但是，这样，我们就真的如各派宗教所说，活一个过程就好，不必追求什么进步与理想了。
    所幸不是真的。那么，我还要玩世不恭吗？佛祖说的好，“于法不说断灭相”。
