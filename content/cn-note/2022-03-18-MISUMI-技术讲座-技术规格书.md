# 规格书的定义

（[阅读原文：MISUMI 日语网站 技術仕様書編 - 1/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1356.html)）

通常买到产品，你会看到它附带一份文件，说明了产品的内容和性能。这种描述产品内容和性能的文件就称为规格书。成熟的工程师或机械设计师要，得能准确地制定规格书，也能正确解释规格书。

## 规格书的内容写什么

**例如汽车**

外形、重量、驱动系统的结构和性能、燃油效率、内饰、颜色、配件、可选项等。

**例如个人电脑**

外形、重量、操作系统、处理速度、内存容量、显示方式、接口、功耗等。

## “仕样书（规格书）”字面解说

- 在日常对话中，有些场景中我们说“しようがない（没办法了）”。当无法改善情况，也无法提供帮助时，口语上就这么说。
- 书面上，“しようがない” = “仕様がない”。
- 在这种情况下，“式样”有两种含义，后者对应着“规格”的含义。
  （一）做事的方式方法；
  （二）对制作物的尺寸、形状、结构等的规定。
- JIS 里的定义
  规格是材料、产品、工具和设备所要求的特定形状、结构、尺寸、成分、能力、精度、性能、制造方法和测试方法等的规定。
  规格书是文档化的规格。
- 规格书的英文 = Specification（简称SPEC：读作“死拜克”），可以指称规格书、设计明细书。

## 商务上的实用定义

综上所述规格书是具体体现缔约方之间订立的合同条款的书面文件，记录了技术上的要求事项，以及实现它们的手段和方法。

# 规格书的种类

（[阅读原文：MISUMI 日语网站 技術仕様書編 - 2/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1357.html)） 

购买大众消费品，消费者在多个商家各种规格和价格中作出选择。但是，对一般公司来说，购买用来生产的设备，（往往没有成熟产品），要将打算购买的内容告知各供应商，在他们的提案之间作比较来选定最终合作者。在社会竞争中，谁能更快地采购到又好又便宜的东西，谁就更有工作专业性优势。

## 从发布规格书到收取规格书的典型购买流程

![Screen Shot 2022-04-15 at 9.31.40 AM](/images/2022-03-18-MISUMI-%E6%8A%80%E6%9C%AF%E8%AE%B2%E5%BA%A7-%E6%8A%80%E6%9C%AF%E8%A7%84%E6%A0%BC%E4%B9%A6/Screen%20Shot%202022-04-15%20at%209.31.40%20AM.png)

1. 采购规格书：买方制作的要购入品的说明书。采购方一般会提供给多家供应商，看谁能做出更便宜、更好用、更快交货的产品，来选择提案。
2. 报价规格书：卖方提出竞标品的报价和内容说明（也可包括示意性的构造图）。此阶段尚未定选供应商，可避免不必要地花费制作工时。此时争的是高性价比：合理的价格、有吸引力的提案（技术可靠）。
3. 最终规格书：被选中的供应商定下交货品的更多细节，作为最终方案的内容说明书（通常包括装配（概念）图等）。作为合同内容，要确保双方都能理解此规格书。如果最终规格书的制定过程出了问题，供应商可能遭受意外损失。

# 规格书的结构

（[阅读原文：MISUMI 日语网站 技術仕様書編 - 3/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1358.html)）

在上一讲中谈到，规格书分三种。第一种就是采购规格书，我们将以此为主题展开说明。

## 采购规格书中列入的项目

需要列出以下项目，其中（1）至（10）项是技术人员的事，（11）至（13）项是采购人员的工作。

| 序号 | 购买规格书中描述的项目 | 备注                                                         |
| ---- | ---------------------- | ------------------------------------------------------------ |
| 1    | 品名                   | 购入品名称                                                 |
| 2    | 用途                   | 使用目的等<br />（例：原型机与量产机大不相同）                     |
| 3    | 货期              | 表示方法<br />a) 订购后的月份 （风险较小的表示法）, <br />b) 年月日 |
| 4    | 交货地点               | 这与交通工具等有关                                        |
| 5    | 规格                   | 记录具体规格项目的部分，是规格书的核心                     |
| 6    | 附件一览               | 随购入品本体一同交付的内容<br /> （例如：图纸、零件清单、操作手册、备件等） |
| 7    | 支给品                 | 由采购方免费提供的物品（例如：支给品、试件等）        |
| 8    | 涂装                   | 指定油漆颜色及涂装方法<br />（如不指定颜色，依对方的方便，厂内设备颜色就不会统一） |
| 9    | 验收条件               | 产品是否按照规格制造的检查方法<br /> （说明：根据设备类型选择验收地点，a）出厂时检查，b）在交货地点安装后检查） |
| 10   | 包装方法              | 描述包装方法（保证），以避免运输和储存过程中的麻烦 <br />（说明：承运人对自动化设备是外行，要注意避免意外事故） |
| 11   | 保证                   | 对交货后出现的故障，要明确说明保修内容和保修期      |
| 12   | 报价条件             | 报价的范围（包括什么）、要提交的文件、报价截止日期等     |
| 13   | 付款条件               | 涉及双方的关系和利益，由采购部门（和财务部门）主导决定 |

供应商会针对所制作的产品，根据上述采购规格书的内容，制定报价规格书。因此，所有影响报价金额的项目采购方都必须明确说明。

> ■ 关于采购规格书碰到的麻烦
>
> 对大型设备，支付外购品的费用有时间差，可能产生不便。
>
> - **需求方**在设备制造完成，确认符合规格性能后才付款。
> - **受注企业**是设计制造厂商，但要组装到设备上的单元 A 是从专业公司采购的。
> - **单元 A 的生产公司**希望在交货后立即回收货款，等不及整体设备完成。
>
> **麻烦在这**
>
> 单元 A 的货款由受注公司处理，但如果 A 单元的付款时间与设备完成后的付款时间差很久，这些货款就成了受注公司的负担。  　
>
> 在这种情况下，承认 a) 付款是受注公司的责任，本着 b) 付款条件用于描述双方可以接受的内容，协商下让 c) 技术担当者决定此单元的检验规格，则可以根据它的性能确认结果来提前支付。如果能把这样的条款写入采购规格书，那就好了。

# 良好的采购规格书

（[阅读原文：MISUMI 日语网站 技術仕様書編 - 4/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1359.html)）

良好的采购规格书，三方（采购方、供货商和制造者）都满意。为此，采购规格书不应对三方造成不必要浪费（如下所述）。

## 不必要的浪费举例

| 序号 | 内容                                             | 举例                                                         |
| ---- | ------------------------------------------------ | ------------------------------------------------------------ |
| ⑴    | 描述不明确，要花费更多会议时间                   | 验收条件不明确<br />规格内容缺失（用途、使用时间等）         |
| ⑵    | 与现有周边设备的连接规范缺失                     | 没有给控制信号作统一的协议，或没有指定必要的品牌型号         |
| ⑶    | 多次发布采购规格变更，得反复变更作成的报价规格书 |                                                              |
| ⑷    | 漏记环境事项，制成后发现问题                     | 无法安装到指定场所<br />空气压力不足                         |
| ⑸    | 外购部品选型问题                                 | 公司内本来有此制品，但却外购了其它公司的商品，事后才要求更换掉它 |

## 良好采购规格书的特质

### ⑴ 采购规格书是订货人向承包人提出的关于制成品的合同

解说

- 承包商必须提出满足采购规格书内容的提案。因此，采购规格书应明确记载可以达成签约的内容。不宜有导致对方误解的内容。
- 对于对方不知道的信息，例如与现有设备共同使用，或使用自有的制造技术，要尽可能详细地披露规格信息，使其作为制造条件要求。
- 不具体表达自己也不清楚的要求，例如，只描述所需性能，让对方选择手段。（让对方肩负起自己的责任，这个态度也很重要。）

### ⑵ 采购规格书明记的内容要让承包商有可能编写报价规规书

解说

- 对于采购规格书的作成者来说，重要的立场是使自己有机会从多个方案中作出选择，即要让每个承包商提出他们擅长的方案。因此，不要用已经知道的内容来覆盖这种可能性。

### ⑶ 悬疑的问题先记下来，声明待补充协议决定

解说

- 用“细节由补充协议决定”等表达方式，明记要很花时间的暂时悬疑问题。

# 作成采购规格书的流程

（[阅读原文：MISUMI 日语网站 技術仕様書編 - 5/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1360.html)）

每个公司都有其组织系统，将工作职能划分到各专业部门。公司购买别公司的产品，最好充分利用每个部门的职能。

## 相关部门及职能

| 组织部门                               | 机能                                                         |
| -------------------------------------- | ------------------------------------------------------------ |
| 生产技术部门                           | - 作成《采购规格书》，详载购入物品生产技术层面内容；<br />- 作成《报价条件明细书》 |
| 使用部门<br />（制造部门、开发部门等） | - 从使用者的角度，讲清要求内容。<br />例如：使用便利性指标、环境要求（安静程度等）、振动条件、使用频率，等等<br />制造部门例：要马上投入使用的设备、不易出故障的设备，等等<br />开发部门例：配置上易作功能改善、容易处理数据，等等 |
| 采购部门<br />会计部门                 | - 在采购政策层面对采购规格书负责<br />例：指定采用的单元，设定支付条件<br />- 选择将采购规格书分发给哪些供应商<br />例：根据购入产品的特点，选择有优势的供应商 |

### 什么是报价条件明细书？

它与采购规格书相应，是提交给承包商的文件之一，明确说明了报价的条件（见以下项目）：
　(1) 产品名称, (2) 数量, (3) 所需交货日期, (4) 交货地点, (5) 用途, (6) 规格(概要), (7)报价范围, (8)提交文件, (9)验收条件 , (10) 保证条款

# 报价规格书的作用

（[阅读原文：MISUMI 日语网站 技術仕様書編 - 6/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1361.html)）

希望得到订单的供应商，收到采购规格书后，必须回复报价规格书（参照下文）和报价单。提交采购规格书意味着参与竞争订单。因此，销售部门是责任部门，技术部门向销售部门提交技术上优越的提案。

报价规格书是什么？满足采购规格书的制品（设备、自动机等）的说明材料。

报价单是什么？报价规格书中制品的价格、货期、关联条件等等，将这些明明确记下来的文件。

## 采购规格书与报价规格书的关系

报价规格书以采购规格书中的要求为根据，但是采购规格书几乎不可能用语言和数值完整表达客户的所有期望。因此，报价规格书试图站在对方的角度，以通俗易懂的方式，表达采购规格书中这些没有完全表达出来的部分，以此来说服对方，这是非常重要的。

例如

- 对象所处行业的技术水平和安全标准
- 采购方的设备采购政策：是倾向于专用机，还是倾向于通用单元的模块化，等等。

# 报价规格书的麻烦事例 （上）

 ([阅读原文：MISUMI 日语网站 技術仕様書編 - 7/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1362.html))

全球化的背景下，自动机的采购方和销售方都出现过各种麻烦。

## 与规格相关的问题增加的因素

与规格书相关的麻烦事增多的因素可以大致分为以下几类：

⑴ 行业构成和企业责任； 

⑵ 制作报价规格书过程中沟通问题； 

⑶ 制定规格书的负责人（生产工程师、销售负责人）能力问题； 

⑷ 采购员能力问题。

但应该注意，其中因素 ⑵“信息共享问题”意外地多。

## 行业构成与企业责任引发的麻烦案例

| 麻烦事例                                                     | 与规格书有关的原因和对策                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| ⑴ 采购规格书里的性能无法实现，交期延迟和人工成本增加，导致亏本<br />⑵ 设计图有误，部品选择有误，等等 | 【原因】<br />组织扁平化、多用外包等等组织体制变化，业务担当者不能作出有经验的判断。<br />【对策举例】<br />重构工作的检查机制。 |
| ⑴ 制造部品、采购部品的实际成本高于预算的报价，导致没有利润<br />⑵ 与对方现有制造设备配合不佳，可用性差 | 【原因】<br />产品品种多，数量少，寿命短，在此趋势下，对规格书未经充分的检查就发出去了。<br />【对策举例】<br />把有类似经验的项目和新规项目分开，在类似项目上注重效率（比如采用家族系列方式）在新规项目上注重聚焦重点课题，更合理地给工程师分配任务。 |
| ⑴ 对重复的规格变更对应不够<br />⑵ 取消订单（收不到罚金）     | 【原因】<br />由于商业规则的崩溃，在收到“买价优先”的订单后，规格发生了变化。<br />【对策举例】<br />- 初期就识别对方品性，规避风险。<br />- 交换违约担保相关文书。 |

除上述情况外，“商业规则崩溃”的例子会有下严重情况：

⑴ 竞争无国界，其他国家公司的商业习惯不同，语言有差异，导致合同出漏洞；

⑵ 采购方预测到产品的生命周期太短，临时撤回订购指令；

⑶ 各家为订单过度竞争，价格崩盘，无视利润。

还有，作成的报价规格书可能被滥用：

⑴ 自己公司作的报价规格书，在其它公司间传阅，成为他们采购规格书的参考；

⑵ 自己公司做出了报价，采购方把“比此价格更低”作为附加条件写入新采购规格书，发布给竞争公司。

# 报价规格书的麻烦事例 （下）

 ([阅读原文：MISUMI 日语网站 技術仕様書編 - 8/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1363.html))

公司内部制作报价规格书时，方法不良也会导致麻烦，现结合案例，解释其原因和对策。

## 制定报价规格书过程中因沟通不畅而引发麻烦的案例

| 麻烦事例     | 与规格书有关的原因和对策   |
| ------------ | ----------------------------------------------------------- |
| ⑴ 包裹不良，导致设备运往海外的过程中生锈。<br />⑵ 试用的设备被用作量产，耐久性不良造成事故。 | 【原因】<br />规格书格式不良，漏记了事项（例：使用用途，运输条件，等）<br />【对策举例】<br />- 确认事项中一定包含需要与对方共同讨论决定的事情。<br />- 规格书要有固定形式，避免遗漏。 |
| ⑴ 交货地点没有明确规定，结果在交货的时候，产生计划外的费用。<br />（例：货物很重，从卡车上搬下来，或者要放到工厂二层，需要用到起重机。）<br />⑵ 设备外周尺寸过大，搬不进去。<br />（例：出入门高，电梯大小，道路宽度等条件没有充分调查，设备制成后才发觉有问题。） | 【原因】<br />公司内各担当沟通不畅，受注后才发觉问题。<br />【对策举例】<br />- 完善规格书的格式，使之没有遗漏，管理者要担当起检查的责任。<br />- 搬运路径要在事前踩点。 |
| ⑴ 过了货期还不能完成设备（不能满足规格），导致耽误对方生产。 | 【原因】<br />决定验收日程的方法有问题（例：识别风险、处理风险）<br />【对策举例】<br />- 判断风险程度，设定交货期。<br />- 事先把握给对方造成的影响，选择制作风险小的设计和制作方法。<br />- 在推进过程中，采用的手段要考虑到给对方造成影响最小化。 |

上面的事故案例，经历过失败就学会了如何处理，但是这种事情给对方和公司都会带来巨大的损失，所以最好通过 OJT （工作全程培训）来防患于未然。

# 规格改动的对策

（[阅读原文：MISUMI 日语网站 技術仕様書編 - 9/9](https://jp.misumi-ec.com/tech-info/categories/machine_design/md01/c1364.html)）

交货期催得越来越急，采购规格难免变化，这会产生许多不利影响。设备设计人员怎样快速响应规格变化，此处提出几个工作要点。

为尽量减少规格变更带来的不利影响，最好在制定规格的阶段，双方就在合同文件中明确说明达成共识的条款。即便如此，规格变更还是会发生，关键是要在“预防”、“预测”和“对策”这三个层面有实用的做法。

### ⑴ 预防：以下应作为合同在规格书中明确说明

- 若采购方提出规格变更，根据内容，可能会发生交货延迟。
- 上述情况下，可能会要求分摊所产生的费用。

### ⑵ 预测：客户和同事都可能图个省心省力，尽早觉察可能产生的规格/设计变更

- 在预测到规格变更的可能性的同时进行处理。

- 尽早感知到规格变更请求，最大限度地减少对本公司的损害。
- 经常与客户担当者、公司相关部门、外包公司联系沟通，尽早发现问题。
- 对以下情况要警惕：
  - 与客户的重要会议（例如最后阶段的规格定稿会议），必要的上级管理人员缺席
  - 翻来覆去的设计审查
  - 内部的设计评审会议，销售人员经常缺席，等等

### ⑶ 对策：规格变还得变，尽可能减小负面影响

- 设计者本人挺身而出，临机应变地采取措施
- 参与做成规格书的销售部门、采购部门、制造部门等的负责人要团队合作。
- 设计师居中要表现出领导力，例如协调人员，汇总信息：
- 对于采购担当——确定下单零部件的进度和变化，指示购入品
  - 对于采购担当——确认部品加工外协厂的进度，提供变更信息
- 对于销售担当——对货期的影响，对制造成本的影响，提供其它信息

以上，MISUMI 技术规格书 九篇完。
