## 参数特点

### 性能参数

| 项目     | 数据     | 解说 |
| -------- | -------- | ---- |
| 工作周期 | 47 秒    |      |
| 打钉时间 | 4 秒     |      |
| 报警率   | < 1/3000 |      |

### 具体特点

- 单制品锁附式及多制品锁附式。
- 可以对每颗的螺钉锁附参数进行单独设定。
- 在实际应用过程中，可以对锁附螺钉的扭力、转角作精准控制。
- 在生产过程中，可以追溯每颗螺钉的锁附结果。
